Ultimate Facebook Page Plugin is a shortcode plugin of New Facebook Page Plugin - Version 2.3

== Description ==

Facebook released it's new version 2.3. Ultimate Facebook Page Plugin Shortcode contains 2.3

Easy and all possible options are available on this plugin.

Options are: Change URL name, Put your facebook url, Hide Cover, Show Faces, Show Streams, Show Small Cover, Width, Height, Adaptive Width.

Yep this plugin is responsive but even if you don't want the responsiveness then you can change.

== Installation ==

This section describes how to install the plugin (Widget) and get it working.

1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

This is a Wordpress Shortcode Plugin.

Shortcode Configuration Fields:

Shortcode ->

To Show Minimal Facebook Display
[ultimatefacebook]

Facebook page URL
[ultimatefacebook fb_url="http://www.facebook.com/FacebookDevelopers"]

If want to Change the URL name
[ultimatefacebook name="Facebook"]

--- name="YOUR_TEXT_INPUT"

If want to Change Width
[ultimatefacebook width="300"]

--- width="INTEGER"

If want to Change Height
[ultimatefacebook height="300"]

--- height="INTEGER"

If want to Hide Cover Photo on top
[ultimatefacebook hide_cover="true"]

--- hide_cover="false"

If want to Hide Faces
[ultimatefacebook show_faces="false"]

--- show_faces="false"

If want to Hide Streams
[ultimatefacebook show_streams="false"]

--- show_streams="false"

If you want to use Small Header

[ultimatefacebook small_header="true"]

--- small_header="true"

If you don't want the facebook module width Adaptive

[ultimatefacebook adapt_width="false"]

--- adapt_width="false"

All together - example

[ultimatefacebook fb_url="http://www.facebok.com/FacebookDevelopers" width="400" height="500" name="TEXT_INPUT" hide_cover="false" show_faces="true" show_streams="true" small_header="false" adapt_width="true"]
