The Array Toolkit extends functionality to Array Themes, providing portfolio items, testimonials and more.

==Description==

**Extend Your Array Theme**

The Array Toolkit provides support for portfolio items, testimonials, galleries, widgets and more to supported Array themes. Simply install the plugin and the theme will determine which features to activate. See your theme's help file to determine whether or not you need the Array Toolkit.

==Frequently Asked Questions==

= What is this plugin and why do I need it? =

The Array Toolkit provides extra functionality to the collection of Array Themes. The Toolkit adds various widgets (Twitter, Flickr, Dribbble and social icons), custom image galleries and a few custom post types. The plugin is not a requirement to use Array themes, but it will extend the themes to function as you see them in the demos.

= Where are the Toolkit settings? =

All options have been automated, and you do not need to change any settings. Features are activated based on their support defined in the theme.

= Can I use this plugin with other themes? =

The Array Toolkit was developed to extend the functionality of Array Themes specifically, however parts of it may be useful to other themes. If using your own theme, you may have to provide some extra styling or modifications to customize it to your needs.

==Installation==

This plugin can be installed directly from your site.

1. Log in and navigate to Plugins &rarr; Add New.
2. Type "Array Toolkit" into the Search input and click the "Search Widgets" button.
3. Locate the Array Toolkit in the list of search results and click "Install Now".
4. Click the "Activate Plugin" link at the bottom of the install screen.
5. The widgets will be available in Appearance &rarr; Widgets.

It can also be installed manually.

1. Download the "Array Toolkit" plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the "Plugins" screen.
4. Locate "Array Toolkit" in the list and click the "Activate" link.
5. The widgets will be available in Appearance &rarr; Widgets.
