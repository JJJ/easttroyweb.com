Promote your latest design work, sketches, audio, photography and more with Designer. Featuring an eclectic, responsive design and a fully-featured blog, you can get a professional portfolio up and running effortlessly, with zero coding required. Designer is easy to customize and ships with several portfolio styles including tiled, landscape, portrait and square.

== Installation ==

1. Sign into your WordPress dashboard, go to Appearance > Themes, and click Add New.
2. Click Add New.
3. Click Upload.
4. Click Choose File and select the theme zip file you downloaded from Array.
5. Click Install Now.
6. Click Add New, then click Upload, then click Choose File.
7. After WordPress installs the theme, click Activate.
8. You've successfully installed your new theme!
